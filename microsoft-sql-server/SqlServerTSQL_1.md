#####  :octocat: [1.声明变量](#top) <b id="target1"></b> 
`概括`:`使用` `Declare` `声明变量 下面为声明变量的语法 可以一次声明一个也可以声明多个`
```sqlserver
Declare @variable_name type [= <value>],
        @variable_name2 type [= <value>]
```
```sql
Declare @age int = 19
```
